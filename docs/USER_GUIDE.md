## 1. 游戏简介

### 1.1 游戏背景
你是农场的守护者！一波波饥饿的野生动物正试图入侵你的农场。建造防御塔楼，布置防线，守护你的庄稼！

### 1.2 游戏目标
- 在敌人到达农场终点前消灭它们
- 保护农场生命值不被耗尽
- 合理规划塔楼位置，抵挡所有波次的进攻
- 完成所有 6+ 个关卡，成为真正的农场守护者！

### 1.3 在线试玩
🎮 **游戏地址**: https://lud0122.github.io/FarmDefense/

## 2. 快速入门

### 2.1 开始游戏
1. 打开游戏页面
2. 点击 **"开始游戏"** 按钮
3. 进入游戏场景

### 2.2 基本流程
1. **获得金币**: 击杀敌人获得金币奖励
2. **选择塔楼**: 点击底部面板或使用数字键 1-6
3. **放置塔楼**: 点击地图空白处放置
4. **抵御进攻**: 塔楼自动攻击范围内的敌人
5. **波次推进**: 击败当前波次后自动进入下一波

### 2.3 胜利与失败
- **胜利**: 完成关卡所有波次，或摧毁所有敌人（Level 5+）
- **失败**: 生命值降至 0（传统关卡）或作物全毁（Level 5+）或所有塔楼被毁（Level 6+）

## 3. 操作指南

### 3.1 桌面端鼠标操作

| 操作 | 说明 |
|------|------|
| 🖱️ **左键点击塔楼按钮** | 选中/取消选中塔楼类型 |
| 🖱️ **左键点击地图** | 放置选中的塔楼 |
| 🖱️ **右键点击塔楼** | 显示回收菜单 |
| 🖱️ **鼠标悬停已放置塔楼** | 显示该塔楼的攻击范围 |
| 🖱️ **选中塔楼后移动鼠标** | 显示攻击范围预览（辅助放置决策） |

### 3.2 桌面端键盘操作

| 按键 | 功能 |
|------|------|
| **WASD** | 控制玩家直升机移动 |
| **R** | 切换玩家直升机射程显示 |
| **1-6** | 快速选择对应塔楼 |
| **ESC** | 取消当前塔楼选择 |

### 3.3 移动端操作

**屏幕左侧 - 虚拟摇杆**:
- 拖动摇杆控制直升机移动
- 直升机会自动攻击范围内的敌人

**屏幕底部 - 塔楼选择面板**:
- 点击选择要放置的塔楼类型
- 再次点击取消选择
- 显示当前金币和塔楼价格

**屏幕右侧 - 操作按钮**:
- ❌ 按钮：取消当前选择
- 👁️ 按钮：切换射程显示

**触屏操作**:
- **点击地图**: 放置选中的塔楼
- **长按塔楼 500ms**: 显示回收菜单

### 3.4 塔楼回收
- **桌面端**: 右键点击已放置的塔楼
- **移动端**: 长按塔楼 500ms
- 点击确认回收，返还 50% 建造成本

### 第 6 关：塔防反击战 🛡️ (终极关卡！)

> ⚠️ **终极挑战**: 敌人获得AI行为，你的塔楼也可能被摧毁！

**配置**
- 起始金币: 💰 1500
- 敌人类型: 全部敌人，带AI行为
- **特殊规则**: 塔楼有生命值，可被敌人攻击

**敌人AI行为**

| 行为类型 | 特点 | 应对策略 |
|----------|------|----------|
| **冲锋型 (Rush)** | 冲向塔楼自爆，造成范围伤害 | 用冰冻塔减速，在到达前消灭 |
| **破塔型 (TowerBreaker)** | 停留攻击塔楼，造成持续伤害 | 分散布置塔楼，不要集中 |

**攻略要点**
| 策略 | 说明 |
|------|------|
| 分散布局 | 破塔型敌人会攻击一个塔很久，分散能降低损失 |
| 前排肉盾 | 高生命值的塔（榴弹塔150HP）放在外围 |
| 及时维修 | 发现塔被攻击时，可以考虑回收重建 |
| 优先击杀 | 先消灭破塔型敌人，保护你的投资 |
| 快速反应 | 冲锋型移动快，需要提前布置冰冻控场 |

**推荐阵型**
```
[机枪]              [机枪]
    [手枪][榴弹][手枪]

[哨戒]   [核心塔]   [哨戒]

    [机枪][冰冻][机枪]
```

**注意事项**
- ❤️ 塔楼生命值一旦归零就会被摧毁
- ♻️ 及时回收受损严重但还没被毁的塔，返还50%金币
- 🚁 直升机在这种情况下特别重要，可以灵活支援薄弱点

## 7. 进阶技巧

### 7.1 塔楼布局策略

#### "减速+输出"组合
```
冰冻塔 → 机枪塔 → 机枪塔 → 榴弹塔
   ↓       ↓         ↓         ↓
 减速    输出      输出     范围收割
```
原理：冰冻塔减速敌人，让输出塔有更多时间攻击

#### "交叉火力"阵型
```
    [机枪塔]
       │
[榴弹塔]─●─[榴弹塔]
       │
    [机枪塔]
```
原理：全方位覆盖，无死角

#### "重点防御"阵型
```
起点 ──→ [冰冻] ──→ [机枪×3] ──→ [榴弹] ──→ 终点
```
原理：把最强火力集中在敌人必经之路

### 7.2 经济管理技巧

1. **开局**: 优先建造便宜的手枪塔和哨戒塔
2. **中期**: 逐步替换成高级塔楼
3. **后期**: 退役边缘位置的低级塔，集中资源强化核心区域
4. **回收**: 及时回收无效位置的塔楼，回收 50% 成本

### 7.3 玩家直升机配合

- 使用 WASD（桌面端）或虚拟摇杆（移动端）灵活移动
- 追着高血量敌人打
- 支援防线薄弱点
- R 键查看射程，保持在最佳位置

**Level 5+ 特别提示**: 直升机成为快速响应单位，随时准备支援被攻击的作物

**Level 6+ 特别提示**: 注意保护外围塔楼，可以用直升机引开冲锋型敌人

### 7.4 攻击范围预览的妙用

**新版本特性**：选中塔楼后，鼠标在地图上移动会自动显示攻击范围

**实战应用**
1. 放置前预览塔能覆盖多少路径
2. 寻找能同时覆盖多条路径的"战略要点"
3. 与其他塔的攻击范围形成重叠，最大化火力密度
4. 避免范围浪费，确保每座塔的效率最大化

### 7.5 Level 6+ 塔楼生命值管理

| 塔楼类型 | 生命值 | 定位 |
|----------|--------|------|
| 榴弹塔 | 150 HP | 前排肉盾 |
| 机枪塔 | 120 HP | 副坦克 |
| 电击塔 | 100 HP | 中排 |
| 手枪塔 | 100 HP | 中排 |
| 哨戒塔 | 90 HP | 后排输出 |
| 冰冻塔 | 80 HP | 需要保护 |

**保护优先级**: 冰冻塔 > 机枪/榴弹塔 > 其他

## 9. 常见问题

### Q: 游戏无法加载？
**A**:
1. 清除浏览器缓存后刷新
2. 确保浏览器支持 WebGL
3. 推荐使用 Chrome、Firefox 或 Edge

### Q: 为什么有些敌人打不到？
**A**:
- 不同的塔有不同的射程
- 检查塔楼是否覆盖敌人行进的路径
- 鼠标悬停在塔楼上查看其攻击范围

### Q: 金币不够用怎么办？
**A**:
- 优先击杀高奖励敌人（熊、野猪）
- 回收无用位置的塔楼
- 升级比新建更划算（如果有升级功能）

### Q: 如何有效对付熊Boss？
**A**:
- 熊血量高达 300，需要集中火力
- 使用冰冻塔减缓其移动速度
- 榴弹塔的范围伤害对熊群效果好
- 机枪塔持续输出
- 玩家直升机集中攻击

### Q: 为什么选中的塔放不了？
**A**:
- 检查金币是否充足
- 点击位置是否在已有塔楼上
- 点击位置是否在路径上（塔楼不能建在路径上，Level 5+ 除外）

### Q: 如何取消当前选中的塔？
**A**:
- 点击已选中的塔楼按钮
- 按下 ESC 键
- 点击已放置的塔楼

### Q: Level 5 的敌人为什么不走路径？
**A**:
**这是设计特性！** Level 5+ 的敌人会直接寻找你的作物，这是新游戏机制：
- 学会在作物周围布防
- 不要只在路径上放塔
- 使用攻击范围预览规划全方位防御

### Q: Level 6 的塔楼怎么会被打坏？
**A**:
**这是新机制！** Level 6+ 的敌人有AI行为会攻击塔楼：
- 分散布置塔楼，降低单个被攻击的风险
- 高生命值塔（榴弹塔、机枪塔）放外围
- 及时回收受损严重的塔，避免完全损失

### Q: 第6关太难了有什么技巧？
**A**:
- **识别敌人类型**: 优先消灭破塔型敌人（会停留在塔边攻击）
- **分散布局**: 不要把所有塔放在一起
- **冰冻控场**: 冰冻塔对冲锋型特别好用
- **灵活应变**: 随时准备回收重建
- **保护核心**: 把冰冻塔和机枪塔放在相对安全的位置
