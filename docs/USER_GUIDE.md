## 1. 游戏简介

### 1.1 游戏背景
你是农场的守护者！一波波饥饿的野生动物正试图入侵你的农场。建造防御塔楼，布置防线，守护你的庄稼！

### 1.2 游戏目标
- 在敌人到达农场终点前消灭它们
- 保护农场生命值不被耗尽
- 合理规划塔楼位置，抵挡所有波次的进攻
- 完成所有 4 个关卡，成为真正的农场守护者！

### 1.3 在线试玩
🎮 **游戏地址**: https://lud0122.github.io/FarmDefense/

## 2. 快速入门

### 2.1 开始游戏
1. 打开游戏页面
2. 点击 **"开始游戏"** 按钮
3. 进入游戏场景

### 2.2 基本流程
1. **获得金币**: 击杀敌人获得金币奖励
2. **选择塔楼**: 点击底部面板或使用数字键 1-6
3. **放置塔楼**: 点击地图空白处放置
4. **抵御进攻**: 塔楼自动攻击范围内的敌人
5. **波次推进**: 击败当前波次后自动进入下一波

### 2.3 胜利与失败
- **胜利**: 完成所有 4 个关卡的所有波次
- **失败**: 生命值降至 0

## 3. 操作指南

### 3.1 鼠标操作

| 操作 | 说明 |
|------|------|
| 🖱️ **左键点击塔楼按钮** | 选中/取消选中塔楼类型 |
| 🖱️ **左键点击地图** | 放置选中的塔楼 |
| 🖱️ **右键点击塔楼** | 显示回收菜单 |
| 🖱️ **鼠标悬停已放置塔楼** | 显示该塔楼的攻击范围 |
| 🖱️ **选中塔楼后移动鼠标** | 显示攻击范围预览（辅助放置决策） |

### 3.2 键盘操作

| 按键 | 功能 |
|------|------|
| **WASD** | 控制玩家直升机移动 |
| **R** | 切换玩家直升机射程显示 |
| **1-6** | 快速选择对应塔楼 |
| **ESC** | 取消当前塔楼选择 |

### 3.3 塔楼回收
- 右键点击已放置的塔楼
- 显示回收菜单（返还 50% 建造成本）
- 点击确认回收

## 7. 进阶技巧

### 7.1 塔楼布局策略

#### "减速+输出"组合
```
冰冻塔 → 机枪塔 → 机枪塔 → 榴弹塔
   ↓        ↓         ↓        ↓
 减速     输出      输出     范围收割
```
原理：冰冻塔减速敌人，让输出塔有更多时间攻击

#### "交叉火力"阵型
```
      [机枪塔]
         │
[榴弹塔]─●─[榴弹塔]
         │
      [机枪塔]
```
原理：全方位覆盖，无死角

#### "重点防御"阵型
```
起点 ──→ [冰冻] ──→ [机枪×3] ──→ [榴弹] ──→ 终点
```
原理：把最强火力集中在敌人必经之路

### 7.2 经济管理技巧

1. **开局**: 优先建造便宜的手枪塔和哨戒塔
2. **中期**: 逐步替换成高级塔楼
3. **后期**: 退役边缘位置的低级塔，集中资源强化核心区域
4. **回收**: 及时回收无效位置的塔楼，回收 50% 成本

### 7.3 玩家直升机配合

- 使用 WASD 灵活移动
- 追着高血量敌人打
- 支援防线薄弱点
- R 键查看射程，保持在最佳位置

### 7.4 攻击范围预览的妙用

**新版本特性**：选中塔楼后，鼠标在地图上移动会自动显示攻击范围

**实战应用**
1. 放置前预览塔能覆盖多少路径
2. 寻找能同时覆盖多条路径的"战略要点"
3. 与其他塔的攻击范围形成重叠，最大化火力密度
4. 避免范围浪费，确保每座塔的效率最大化

## 9. 常见问题

### Q: 游戏无法加载？
**A**:
1. 清除浏览器缓存后刷新
2. 确保浏览器支持 WebGL
3. 推荐使用 Chrome、Firefox 或 Edge

### Q: 为什么有些敌人打不到？
**A**:
- 不同的塔有不同的射程
- 检查塔楼是否覆盖敌人行进的路径
- 鼠标悬停在塔楼上查看其攻击范围

### Q: 金币不够用怎么办？
**A**:
- 优先击杀高奖励敌人（熊、野猪）
- 回收无用位置的塔楼
- 升级比新建更划算（如果有升级功能）

### Q: 如何有效对付熊Boss？
**A**:
- 熊血量高达 300，需要集中火力
- 使用冰冻塔减缓其移动速度
- 榴弹塔的范围伤害对熊群效果好
- 机枪塔持续输出
- 玩家直升机集中攻击

### Q: 为什么选中的塔放不了？
**A**:
- 检查金币是否充足
- 点击位置是否在已有塔楼上
- 点击位置是否在路径上（塔楼不能建在路径上）

### Q: 如何取消当前选中的塔？
**A**:
- 点击已选中的塔楼按钮
- 按下 ESC 键
- 点击已放置的塔楼

### Q: 第4关太难了有什么技巧？
**A**:
- 这关确实是高难度挑战
- 充分利用攻击范围预览功能规划布局
- 前4波尽量保留一些金币
- 第5波开始前确保你的阵型完整
- 灵活使用直升机支援最危险的位置
- 不要害怕回收重建
